# MenuingOfMemory (Re:Fined module)

Small Re:Fined-compatible module that patches KH2 menu cursor behavior to
improve submenu cursor handling and provide a toggleable mode (Off / Magic-only / All).

## Installation
Install via OpenKH Mod Manager.
```
skylect-dev/MenuingOfMemory-ReFined
```

## Usage
- Toggle modes in-game using the button combo: hold L2 + R2 then press D-Pad Up/Down.
- The module shows a short informational popup when the mode changes.
- Settings are saved to `MenuingOfMemory.ini` next to the DLL.

## Compatibility
- Built for Re:Fined. 
- Do NOT run the standalone and the Re:Fined module simultaneously!
